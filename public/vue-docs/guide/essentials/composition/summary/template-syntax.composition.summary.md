- Vue uses an HTML-based template syntax for data binding
- Templates are compiled into optimized JavaScript code
- Text interpolation is done using double curly braces (`{{ }}`)
- Raw HTML can be rendered using the `v-html` directive
- Attribute bindings are done using the `v-bind` directive or its shorthand (`:`)
- Boolean attributes can be bound using `v-bind` with a truthy value
- Multiple attributes can be bound using `v-bind` without an argument
- JavaScript expressions can be used in data bindings
- Directives are special attributes with the `v-` prefix
- Directives can have arguments and modifiers for special behavior
